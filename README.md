# 🚀 Monad 空投批量归集工具 (Monad Sweeper App)

这是一个专门为 Monad 主网空投设计的前端 DApp，用于帮助用户安全、高效地将多个空投钱包中的 $MON 代币归集到一个中心化交易所或个人地址。

该工具基于现代 Web3 技术栈构建，旨在提供简洁、安全的批量操作体验。

## ✨ 核心功能

* **批量归集：** 支持从多个 EVM 钱包地址（通过私钥输入）批量发起交易。
* **灵活的转账模式：** 支持“归集所有余额（减去 Gas 费）”和“转账固定金额”两种模式。
* **私钥解析：** 智能解析多种输入格式 (`私钥 金额` 或 `私钥,金额`)。
* **安全提示：** 明确警告用户私钥操作的风险，引导用户在安全/离线环境中使用。

## 🛠️ 技术栈

* **前端框架:** [Next.js](https://nextjs.org/) (使用 App Router 和 Static Export)
* **样式:** [Tailwind CSS](https://tailwindcss.com/)
* **Web3 交互:** [Wagmi](https://wagmi.sh/) / [Viem](https://viem.sh/) (处理钱包连接、状态管理和链上交易)
* **语言:** TypeScript / React

## ⚠️ 安全警告

**本项目要求用户输入钱包私钥。请务必在以下环境中操作：**

1.  **离线计算机 (Offline Environment):** 最好在没有连接网络的计算机上操作，以最小化私钥泄露的风险。
2.  **仅用于小额或空投钱包:** 避免使用存有大量资金的主钱包私钥。
3.  **理解风险:** 本工具的开发目的是为了方便用户，但链上归集操作本身存在 Gas 波动、交易失败等风险，请谨慎使用。
